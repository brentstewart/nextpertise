Simple Apache Setup on Linux

by Brent Stewart on Monday, Aug 30, 2021

I’ve been running the free version of ESXi 6.5 for a while, but the vulnerabilities kept piling up and I had issues upgrading it. I’ve been looking at ProxMox VE for a while and last week I decided to give it a chance. I’ll write about Proxmox another time, but I’m building out my local infrastructure on ProxMox and wanted to start by walking through a simple Apache website setup.

First, a disclaimer: this isn’t a public site, it’s intended for internal use as a landing page when the browser starts.

I built an Ubuntu 21.04 Server VM on proxmox. My intention is to use this machine for a variety of services. For this first piece, I installed a webserver. At this point you should be able to browse to http://127.0.0.1 and see the default Apache page. Ubuntu 21.04 has the firewall up by default, so I needed to allow the Apache.

sudo apt update
sudo apt install apache2
sudo ufw allow 'Apache'

I created a directory for my site user at /var/www/stewart and copied my simple start page . Then I and then created a profile for my new site.
