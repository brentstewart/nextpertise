Cockpit
Brent Stewart | Saturday, May 1, 2021
If “web-based graphical interface for administering Linux” makes you think Webmin, then you need to look at Cockpit. Cockpit is a modern web-based management tool for all your Linux servers. It’s similar to webmin, but slicker. Installation in Ubuntu is a breeze. I installed this on all my linux machines and can manage all of the systems from one dashboard (again, Cockpit has to be installed on them all). Cockpit is the base install, with other packages to add functionality for storage, networking, pack manager, virtual machines, and containers.
